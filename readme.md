# Express Server with MongoDB Integration

This is an Express server application that connects to a MongoDB database and serves data through a REST API.

## Features

- Connects to MongoDB using MongoDB Atlas
- CORS enabled for cross-origin requests
- JSON parsing middleware for handling request bodies

## Prerequisites

Before you begin, ensure you have met the following requirements:

- [Node.js](https://nodejs.org/) (Recommended: v16 or higher)
- [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) account (for database connection)
